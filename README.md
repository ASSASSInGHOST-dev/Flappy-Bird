# Flappy-Bird



🐦 Flappy Bird Clone (Python)

A simple Flappy Bird-style game built using Python. This project demonstrates game development concepts like physics simulation, collision detection, and event handling.

🎮 Features

Smooth bird movement with gravity
Pipe generation and scrolling
Collision detection
Score tracking system
Game over and restart functionality

🛠️ Tech Stack

Python
Pygame
